# Политика конфиденциальности Fern

_Обновлено: 13 июля 2026 г._

Fern — офлайн-приложение для изучения языков. У него нет аккаунтов, нет сервера
и нет аналитики. Ниже — что именно происходит с данными.

## Что Fern собирает

**Ничего.** Разработчик не получает ни ваших слов, ни статистики, ни
идентификаторов устройства. Аккаунт не нужен, регистрации нет.

## Где хранятся ваши данные

Всё — только на вашем устройстве: колоды, карточки, история повторов, книги и
транскрипты видео, настройки. Автоматический бэкап (раз в сутки) пишется в
приватную папку приложения на том же устройстве.

Системное резервное копирование Android (Google Drive Auto Backup) для Fern
**отключено**: данные не уезжают в облако Google даже неявно. Перенести словарь
на новый телефон можно только вручную — экспортом в файл (Настройки → Резервная
копия), и этот файл принадлежит вам.

Удаление приложения удаляет все данные. Кнопка «Удалить все данные» в настройках
делает то же самое, не удаляя приложение.

## Когда Fern выходит в сеть

Приложение работает офлайн. Сеть используется только для действий, которые вы
запускаете сами:

| Действие | Куда уходит запрос | Что уходит |
|---|---|---|
| Проверка обновлений | GitHub (`api.github.com`) | Только запрос последнего релиза |
| Перевод слова / билингва | Google Translate, DeepL или **ваш** сервер (Ollama, LibreTranslate, любой OpenAI-совместимый) — на ваш выбор в настройках | Слово или фраза, которую вы переводите |
| Словарная справка | `dictionaryapi.dev`, Wiktionary | Запрашиваемое слово |
| Разбор видео | YouTube | Ссылка на видео (для субтитров и звука) |
| Импорт статьи по ссылке | Сайт, чью ссылку вы дали | Сам запрос страницы |

Никакие из этих запросов не содержат идентификаторов вас или устройства.
Если вы не пользуетесь переводом и видео, Fern не обращается в сеть вовсе
(кроме проверки обновлений, которую можно просто игнорировать).

Обычный (незашифрованный) HTTP разрешён только для адресов в локальной сети —
чтобы работал ваш собственный сервер перевода. Весь остальной трафик идёт по
HTTPS.

## Распознавание текста и офлайн-перевод

OCR (текст с фото) и офлайн-перевод работают на устройстве через Google ML Kit.
Снимки и распознанный текст никуда не отправляются. При этом сама библиотека
ML Kit — часть экосистемы Google Play Services и может отправлять Google
техническую диагностику о своей работе; на это разработчик Fern повлиять не
может. Подробности — в политике Google:
https://policies.google.com/privacy

## Разрешения

- **Камера / галерея** — только когда вы сами выбираете фото для распознавания текста.
- **Уведомления** — напоминание позаниматься, если вы его включили.
- **Установка пакетов** — обновление приложения из GitHub (sideload-версия).
- **Интернет** — см. таблицу выше.

## Дети

Fern не предназначен специально для детей и не собирает данные ни у кого.

## Изменения

Если политика изменится, обновлённая версия появится здесь же, вместе с датой.
