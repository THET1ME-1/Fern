# Fern — разбор рынка и конкурентов

Актуально на 2026-07-02. Источники — форумы Anki, HackerNews, App Store, RuStore, агрегаторы цен, GitHub.

## Прямые конкуренты

### Anki / AnkiDroid
- Алгоритм эталонный, но **UX из 2013-го**, ~90 минут только на освоение.
- **FSRS — opt-in, НЕ дефолт** (тикет #3616 «сделать дефолтом» открыт). FSRS-6 в Anki 25.07 / AnkiDroid v2.22.2.
- Цены: desktop + AnkiWeb-синк + AnkiDroid — бесплатно, open-source. **iOS AnkiMobile — разовые $24.99**.
- **Синк-катастрофы**: первый синк идёт в одну сторону и затирает данные. Форум завален «I lost my cards after sync».
- AnkiDroid активно переделывает UI в 2025-2026 (v2.23.0 — новый экран изучения) — окно «мы красивее» сужается, торопиться.

### Quizlet
- **1 авг 2022**: режимы Learn и Test заперты за подписку. Сейчас бесплатно — ~20 вопросов Learn/мес, 3 теста/мес.
- Массовый отток к **Knowt** и Anki, призывы «boycott Quizlet».
- Цены: **Quizlet Plus $35.99/год или $7.99/мес**.
- В **RuStore**: 4,5★, но оплата гонит в **Google Pay (в РФ не работает)**, вылеты, пропажа терминов при редактировании.

### Duolingo
- **Своих колод нет** — фиксированный курс. Отдельное флеш-приложение Tinycards закрыли в 2020.
- Полезен только как референс по геймификации/удержанию.
- Super ~$84–96/год; **Duolingo Max — $29.99/мес**.

### Memrise
- SRS формально есть, но **вырезаны пользовательские курсы и мнемоники** (2022–2023). Ушли в видео + AI-болталку MemBot.
- Ветераны обижены («пощёчина»).

### Brainscape / Mochi / RemNote
- **Brainscape** — Confidence-Based Repetition, Pro $7.99/мес годовой. Дорого помесячно.
- **Mochi** — markdown, local-first, красивый; free без синка (одно устройство), Pro $5/мес.
- **RemNote** — заметки + SRS (SM-2 дефолт, FSRS бета), free с лимитами, Pro $8/мес. Жалобы: медленно, глючный синк.

### AI-генераторы карт (2025-2026) — «текст/PDF → колода» стало стандартом
- **Knowt** — лидер роста: **5M+ пользователей**, дал бесплатно то, что Quizlet спрятал.
- StudyFetch, Rember (FSRS + MCP-интеграция с Claude), StudyGlen (OCR + FSRS + 37 языков) и др.
- **Щель**: качество AI-карт ругают — «оптимизированы под количество, а не качество припоминания». Ниша — AI, делающий *хорошие атомарные* карты.

## Российский рынок / RuStore
- ~498 приложений по запросу «карточки», но живых мало.
- Anki (4,7★) и Quizlet (4,5★) присутствуют, у обоих проблемы (см. выше).
- Локальные: ReWord (4,8★, 100k+ — заточен под готовый словарь), Skyeng, Lingualeo, Lexilize, SWOTWords, MemoWord — все маленькие, с UX-шероховатостями.
- **Пустота**: полированный офлайн-first русскоязычный SRS для *своих* колод с чистым UX и рабочей оплатой.
- Сквозные жалобы (= точки входа): агрессивная полноэкранная реклама с крошечным крестиком; **нет разовой покупки «убрать рекламу»**; потеря данных при редактировании; сломанная оплата.
- Монетизация: реклама Яндекса (РСЯ) — стандарт; RuStore Billing комиссия **0,9–3,35%** (против ~15%); RuStore к концу 2025 — 65,5 млн MAU.

## Алгоритм: FSRS vs SM-2
- FSRS-6 = 21 параметр, де-факто современный стандарт (~20-30% меньше повторов при 90% удержании против SM-2).
- Реализации разрешительные: py-fsrs (MIT), fsrs-rs (BSD-3).
- **Dart-порт готов**: `fsrs` на pub.dev v2.0.1 (MIT) — только планировщик, без оптимизатора.
- FFI к fsrs-rs через flutter_rust_bridge — оправдан только ради оптимизатора (позже).
- **Решение: FSRS через pure-Dart `fsrs` на дефолтных весах.**

## Технические возможности без опекса
- **Офлайн-перевод**: `google_mlkit_translation` v0.13.1 (MIT). Бесплатно, офлайн после загрузки моделей, 50+ языков, ~30 МБ/язык. Android+iOS. Модели англо-центричны.
- **Офлайн-TTS**: `flutter_tts` v4.2.5 (MIT). Нативные движки устройства. Офлайн если установлен голосовой пакет языка — набор зависит от OEM.
- **Импорт .apkg**: ZIP + SQLite (`collection.anki2`/`anki21` — простой SQLite; `anki21b` с Anki 2.1.50 — zstd + protobuf-маппинг медиа). Зрелых Dart-парсеров нет — своя реализация. Референсы: Node anki-apkg-parser, Python ankisync2.

## Незакрытые ниши 2026
- Уродливый UI Anki — живой рынок аддонов-косметики (люди доплачивают за красоту).
- Создание карт — боль №1 («выделил в браузере/PDF → упало в inbox → ревью» почти нет на мобиле).
- Синк-катастрофы Anki — простор для «синка без выстрела в ногу».
- **Кейс Noji (ex-AnkiPro)**: паразитировал на бренде, брал подписку за обычный SM-2, «офлайн» не работал, серверы легли на 10 дней, на просьбы восстановить данные рикроллил. Вывод блогера ≈ ТЗ: «demand exists for a genuinely modern, trustworthy Anki-compatible client with reliable syncing and honest feature claims».
- **Мобильный майнинг** (словарь + TTS + добавление карты из читалки/субтитров/OCR без десктоп-setup) — самый липкий недозакрытый сегмент.

## Честное предупреждение
«Красивый Anki вообще» перегрет (Mochi, RemNote, Noji, Brainscape, StudyGlen + open-source на Flutter; «beautiful + FSRS + AI + apkg-import» почти коммодитизирован). Выигрывает **пересечение** клиньев, а не каждый по отдельности — особенно связка Material You + офлайн-доверие + русский UX + рабочая оплата RuStore, где вся инфраструктура уже построена под Togetherly.
